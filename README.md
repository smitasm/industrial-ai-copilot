# Industrial AI Copilot

An enterprise-grade AI Copilot built using:

- Python
- LangChain
- ChromaDB
- HuggingFace Embeddings
- Ollama
- Phi-3 Mini

## Current Features

- PDF ingestion
- Semantic chunking
- Vector database
- Semantic retrieval
- Local LLM (Ollama)
- Retrieval-Augmented Generation (RAG)

> This project is being built incrementally following production AI engineering practices.
